# 有熊并击输入框架
解字成根、并击输根、以根输字。

## 特点
- 纯单字、无字库。
- 左右对称、一击可打双字根。
- 简码键。

## 可更换
- 解字方案
- 并击系统
- 按式映射
- 简码生成法
- 单字频率

## 生成字典
[Make](https://www.gnu.org/software/make/manual/make.html)可用来生成字典、生成的文件会被放入build/。
